# 纪念册

让我们一起去看世界，走遍天涯海角，体验世间繁华！

## 网站架构

### 一期

一期网站直接生成静态页面，数据源全部为静态资源，上传到 Github

页面结构

- 登录
  - 密码先写死
  - 简单登录表单
  - 浪漫背景图
- 纪念日页面
  - 炫酷浪漫的动效（选一）
    1. 雪花飘落，冰墩墩逐渐出现，桃心花瓣逐渐出现，计时器渐显，想说的话逐渐出现
    2. 春意盎然的背景图，开花结果，果实为桃心装，桃心内计时器渐显，想说的话逐渐出现
  - 播放结束后出现弹窗，引导去查看历程
- 历程（时间轴）
  - 列表
  - 详情
  - 浪漫版历程播放器
